Posted by Laura Joneson Monday, June 21st 2027.


What is HTML?

HTML dtands for HyperTextMarkup Language. it's a markuplanguage that web
developers use to structure and describe the content of a web page (not
a programming language).

HTML consists of elements that describe different types of content: paragraphs,
links, heading, images, video, etc. Web browsers understand HTML and
render HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element
